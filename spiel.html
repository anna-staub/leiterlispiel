Spielregeln

Es treten 2 Spieler gegeneinander an, die abwechselnd zum Zug kommen. Mit einem Klick auf den Würfel-Button, wird mit einem gewöhnlichen 6-seitigen Spielwürfel gewürfelt. Beim Klick auf den Spezialwurf-Button können die Zahlen -5 bis 10 gewürfelt werden. Man kann also mit einem Wurf weiter kommen, muss aber eventuell auch zurückfahren oder bleibt bei einer 0 auf dem aktuellen Feld stehen. Der Spieler am Zug darf um die gewürfelte Anzahl auf den nummerierten Feldern vor (oder zurück) fahren. Wird eine 6 gewürfelt, darf der gleiche Spieler nach Abschluss des Zuges erneut würfeln. Landet ein Spieler auf einer blauen Röhre fällt er um einige Felder zurück, auf einer rosaroten Röhre wird er um einige Felder weiter transportiert. Bei Feld 55 handelt es sich um ein Spezialfeld. Landet man darauf, kann man wählen, ob man seine Position mit derjenigen des Gegners wechseln oder dort bleiben möchte. Das Zielfeld muss genau getroffen werden. Wird eine zu hohe Zahl gewürfelt, muss um die überzählige Anzahl Felder zurückgefahren werden. Der Spieler der zuerst direkt auf dem Zielfeld landet, gewinnt.
